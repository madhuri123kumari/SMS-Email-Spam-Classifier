# 📧 SMS Spam Classifier using Machine Learning

📌 Project Overview
The SMS Spam Classifier is a Machine Learning project that automatically classifies SMS messages as Spam or Ham (Not Spam). The system uses Natural Language Processing (NLP) techniques and Machine Learning algorithms to identify unwanted messages and improve communication security.

🎯 Problem Statement
Spam messages are a common issue that can lead to fraud, phishing attacks, and unnecessary distractions. This project aims to build an intelligent model that can accurately detect spam messages and filter them automatically.
